# ERP Roadmap v1

## Phase 1

- Build ERP master tables for customers and products
- Build inquiry to quote flow
- Build quote to order conversion
- Build basic sales and inventory foundation
- Keep existing LINE bot tables running in parallel

## Phase 2

- Add warehouse and stock movement workflows
- Add shipment and invoice operations
- Build customer 360 page
- Build order and sales dashboards

## Phase 3

- Add approval flow and permissions
- Add procurement workflow
- Add accounting export and reconciliation
- Add AI-assisted ERP operations

## Suggested first implementation order

1. `erp_customers`
2. `erp_products`
3. `erp_quotes`
4. `erp_quote_items`
5. `erp_orders`
6. `erp_order_items`
7. `erp_sales`
8. `erp_sale_items`
9. `erp_inventory_balances`
10. `erp_inventory_transactions`
